API Doorprize

API sistem undian doorprize dengan integrasi Supabase. Saat ini API sedang berjalan dan siap digunakan.

GET /api/
API info dan health check
GET /api/health
Detailed health check dan environment info
GET /api/vouchers
Ambil semua voucher
POST /api/vouchers
Buat voucher baru (memerlukan autentikasi x-api-key)
